Free Design Requirements


1) Use of positioning schemes

  • Sticky: CSS folder -> overall.css: line 41
  • relative: CSS folder -> information.css: line 35

2) Apply two different kinds of CSS3 techniques

  • round corner: CSS folder -> apply.css: line 8, line 145, line 178; ...
  • gradient: CSS folder -> overall.css: line 25, line 79
  • box-shadow: CSS folder -> apply.css: line 149, line 174; visit.css: line 70, line 90; ...
  • transition: CSS folder -> visit.css: line 78; overall.css: line 62; ...
  • flex: CSS folder -> visit.css: line 16
  • animation: CSS folder -> overall.page: line 99-108

3) how styles are grouped into style sheet(s)/embedded style

  • The CSS folder contains all the stylesheet files. Each CSS file is named similarly to its corresponding HTML file
  • The overall.css file is used to define the common styles applied across all pages, including the header, footer, font family, font size, and other global layout settings.
  • All styles are applied using external style sheets
  • No embedded style
